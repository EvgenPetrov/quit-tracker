# 🚭 Quit Tracker — Трекер без сигарет

Простой трекер отказа от курения с напоминаниями в Google Календарь.

Без рекламы. Без подписок. Без регистрации.

## Что умеет

- ⏱ Живой счётчик времени без сигарет
- 💰 Подсчёт сэкономленных денег
- 🚬 Счётчик невыкуренных сигарет
- ❤️ Прибавка к жизни (каждая сигарета = −11 минут)
- 📅 Напоминания на каждую веху прямо в Google Календарь
- 📱 Работает на телефоне и компьютере

## Как пользоваться

1. Откройте [страницу трекера](https://evgenpetrov.github.io/quit-tracker/)
2. Введите дату отказа, цену пачки и количество сигарет в день
3. Нажмите «📅 Напомнить» на каждой вехе — событие добавится в Google Календарь
4. Готово! Напоминания придут на телефон в нужный день

Ваши данные сохраняются **только в вашем браузере** (localStorage). Никакие данные никуда не отправляются.

## Деплой на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите файл `index.html` в корень репозитория
3. Перейдите в **Settings → Pages**
4. В разделе **Source** выберите ветку `main` и папку `/ (root)`
5. Через минуту сайт будет доступен по адресу `https://YOUR_USERNAME.github.io/quit-tracker/`

## Вехи здоровья

| Срок     | Что происходит                    |
| -------- | --------------------------------- |
| 1 день   | Пульс и давление нормализуются    |
| 3 дня    | Никотин выведен из организма      |
| 1 неделя | Лёгкие начинают очищаться         |
| 1 месяц  | Функция лёгких +30%               |
| 1 год    | Риск болезней сердца ↓ вдвое      |
| 5 лет    | Риск рака лёгких ↓ вдвое          |
| 10 лет   | Риск рака лёгких как у некурящего |
